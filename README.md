# TCP server and TCP client.


## TCP server:
Create a Qt console application project and choose a new C++ class name as server.<br>
.pro: <br>
  QT += network <br>
  server.h:<br>
  #include <QTcpServer>  
  #include <QTcpSocket>  
  #include <QVector>  
  #include <QDebug>  
  
#include <QTcpServer>
#include <QTcpSocket>
